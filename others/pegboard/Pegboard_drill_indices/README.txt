# Summary

I've been trying to make better use of my pegboards. These drill indices will get a lot of drill bits out of drawers and off the work bench. I print them rotated 90 degrees so the holes are pointing upwards. These are sized for 1" on center, 1/4" diameter holes.  

Print the label sheet and cut the label strips apart. I glued them in with Elmer's.  

You'll need one pair of pegboard connectors per index. The connectors mate with the 'ears' on the bottom of each index.
